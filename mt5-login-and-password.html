MT5 Login and Password Help

MT5 Login

  1. Open the MT5 program.
  2. Go to File.
  3. Select “Login to Trade Account”.
  4. Enter your trading account number.
  5. Enter your trading password.
  6. Select the correct server from your account email.

Important

If you log in using an investor password, you will not be able to trade. Investor credentials are view-only.

Website Password Reset

  1. Go to Login.
  2. Click “Forgot password”.
  3. Enter your active email address.
  4. Follow the password reset instructions sent by email.

MT5 Password Reset

  1. Log in to the website.
  2. Go to your MT5 account section.
  3. Click the top-right settings icon.
  4. Select the password change option.
  5. Set a new password and confirm it with the email code.

Password Format

Password must include uppercase and lowercase English letters, numbers, and special characters, with at least 8 characters.
